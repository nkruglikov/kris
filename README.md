# kris

`kris` is a CLI tool for interaction with Christofari.

## Installation
```bash
pip install --force-reinstall git+https://gitlab.com/chit-chat/kris.git
```

## Usage
See `kris --help` for details.

## Development
```bash
git clone git+https://gitlab.com/chit-chat/kris.git
cd kris
pip install -e .
```
